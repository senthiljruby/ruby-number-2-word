## Getting Started

Getting up and running is as easy as 1, 2, 3.

1. Make sure you have [Ruby](https://www.ruby-lang.org/en/) installed.
2. Navigate the path,

    ```
    cd path/to/ruby-number-2-word
    ```

3. Start your app

    ```
    ruby index.rb
    ```
